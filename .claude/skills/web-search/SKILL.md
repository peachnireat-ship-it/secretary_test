## 검색 전략

### 쿼리 공식화

주제 하나당 3~5개 쿼리를 변형하여 검색한다:
- **공식 표현**: 정확한 용어 (예: "인공지능 규제 EU AI Act")
- **시사 표현**: 뉴스에서 쓰는 표현 (예: "AI 규제 법안 최근 동향")
- **영어 병행**: 한국어 결과 부족 시 영어 쿼리 추가
- **최신 필터**: 쿼리에 연도 또는 "최신" 추가

### 소스 우선순위

1. 공식 기관·정부 사이트 (신뢰도 최고)
2. 주요 언론사 (Reuters, BBC, 연합뉴스, 조선/중앙/동아 등)
3. 전문 분석 기관·싱크탱크
4. 개인 블로그·SNS (의견 참고용, 사실로 처리 금지)

### 소스 신뢰도 판단

- 저자·기관 명시 여부 확인
- 게시일 확인 (오래된 자료는 "구 정보"로 표시)
- 광고성·홍보성 글 제외

## 출력 형식

```markdown
# 웹 조사 결과: {주제}
조사일: {날짜}

## 핵심 발견
- [발견1] — 출처: [URL] ({날짜})
- [발견2] — 출처: [URL] ({날짜})

## 주요 뉴스/공지
| 제목 | 날짜 | 핵심 내용 | URL |
|------|------|-----------|-----|

## 전문가 의견
- {이름/기관}: "{의견 요약}" — 출처: [URL]

## 조사 범위
- 사용한 쿼리: ...
- 접근 불가 소스: ... (없으면 "없음")
```
