## SEO 최적화 전략

### 제목 최적화

**제목 구조 원칙:**
- 60자 이내 (모바일에서 잘리지 않는 한계)
- 핵심 키워드를 앞 30자 안에 배치 (알고리즘 가중치 + 시선 패턴)
- 숫자 활용: "5가지", "3분 안에", "2025년 최신" — 구체성이 CTR을 높임
- 괄호/대괄호: [실전편], [초보 필수] — 타겟 시청자를 명확히 함

**제목 후보 5개 작성 기준:**
각 후보는 서로 다른 접근법을 사용해야 한다:
1. 검색어 최적화형 (키워드 그대로 포함)
2. 혜택/결과 강조형 ("~하면 ~된다")
3. 호기심 자극형 ("~인 이유", "~의 비밀")
4. 타깃 명시형 ("초보자를 위한", "직장인 필수")
5. 숫자+키워드형 ("N가지 {키워드}")

클릭베이트 금지: 내용과 무관한 제목은 시청 완료율을 해쳐 알고리즘 페널티를 유발한다.

### 설명란 최적화

```
[첫 2줄 — 펼치기 전 노출되는 핵심 영역]
{핵심 키워드 포함 + 영상이 주는 가치 요약 1~2문장}

[타임스탬프]
00:00 인트로
00:30 {섹션 1}
...

[관련 리소스]
- {링크 또는 설명}

[채널 소개 + 구독 유도]
{채널 한 줄 소개}
구독: {링크}
```

타임스탬프가 있으면 유튜브가 챕터를 자동 생성해 검색 노출과 시청 경험 모두 향상된다.

### 태그 전략

태그는 15~20개, 세 레이어로 구성:
1. **정확히 일치 키워드** (3~5개): 영상 주제 그 자체 — 예: "AI 생산성 툴", "ChatGPT 활용법"
2. **상위 카테고리 키워드** (5~7개): 더 넓은 주제 — 예: "AI", "생산성", "업무 효율"
3. **관련 검색어** (5~8개): 시청자가 다른 방식으로 검색할 표현 — 예: "AI 업무 자동화", "ChatGPT 사용법 초보"

태그는 알고리즘의 주제 파악에 도움을 주지만, 제목과 설명보다 가중치가 낮다. 완성도 있는 제목/설명이 우선이다.

## 출력 형식

```markdown
# SEO 패키지: {주제}
작성일: {날짜}

## 제목 후보

| # | 제목 | 유형 | 길이 |
|---|------|------|------|
| 1 | ... | 검색어형 | N자 |
| 2 | ... | 혜택형 | N자 |
| 3 | ... | 호기심형 | N자 |
| 4 | ... | 타깃형 | N자 |
| 5 | ... | 숫자형 | N자 |

**추천 제목**: {번호} — 이유: {한 줄 근거}

## 설명란 (복사 바로 사용 가능)

{설명란 전문}

## 태그 목록 (쉼표 구분)

{태그1}, {태그2}, ...

## 핵심 키워드 메모

(thumbnail-designer 참조용)
- 주요 키워드: ...
- 감정 톤: ...
```
