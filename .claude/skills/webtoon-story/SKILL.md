## 목적

웹툰 에피소드의 스토리를 장면(Scene) 단위로 분해하고, 창작팀(캐릭터·패널·대사)이 일관된 세계관에서 작업할 수 있는 스타일 시트를 함께 생성한다.

## 핵심 원칙

### 장면 분할 기준

스토리는 산문이 아니라 **장면(Scene)** 단위로 작성한다. 각 장면은 다음 조건을 만족해야 한다:
- 장소가 바뀌거나 시간이 경과하면 새 장면
- 감정의 전환점이 발생하면 새 장면
- 패널 플래너가 이 장면만 보고도 레이아웃을 잡을 수 있어야 함

### 패널 힌트 포함

스토리 작성 시 각 장면에 패널 힌트를 포함한다. 힌트는 권고이지 명령이 아니다:
- `풀 패널 권장` — 감정 고점, 충격, 압도적 장면
- `클로즈업 권장` — 감정 표정이 핵심인 장면
- `분할 패널 권장` — 두 캐릭터 대비, 빠른 템포
- `서술 패널` — 내레이션 위주, 시간 경과

### 스타일 시트 생성 규칙

**반드시 스토리와 함께 `00_style_sheet.md`를 생성한다.** 스타일 시트는 창작팀의 공통 언어다. 없으면 캐릭터 디자이너가 작업을 시작할 수 없다.

스타일 시트 필수 항목:
1. **세계관 톤** — 분위기, 색온도, 장르 감성 키워드
2. **캐릭터별 스타일 가이드** — 외모·의상·표정·말투 키워드
3. **비주얼 언어 가이드** — 감정 고조/회상/충격/코믹 장면의 처리 방향
4. **선 스타일 방향** — 두께, 배경 밀도, 효과선 사용 여부

### 감정 곡선 설계

감정 곡선은 에피소드의 호흡이다. 패턴:
- **기승전결**: 안정 → 불안 → 절정 → 해소
- **복선 회수**: 복선(초반) → 오해 심화 → 회수(클라이맥스)
- **반전**: 평온 → 균열 → 반전 충격 → 여운

에피소드가 길면 소기승전결이 2~3회 반복된다.

## 출력 형식

**`_webtoon/{slug}/00_style_sheet.md`:**

```markdown
# 스타일 시트
작품명: {이름}  에피소드: {번호}

## 세계관 톤
분위기: ...
색온도 방향: ...
장르 감성 키워드: ...

## 캐릭터별 스타일 가이드
| 캐릭터 | 외모 키워드 | 의상 방향 | 표정 특징 | 말투 특징 |
|--------|-----------|---------|---------|---------|

## 비주얼 언어 가이드
- 감정 고조 시: ...
- 회상 장면: ...
- 충격/반전: ...
- 코믹 장면: ...

## 선 스타일 방향
두께: / 질감: / 배경 밀도: / 효과선 사용:
```

**`_webtoon/{slug}/01_story.md`:**

```markdown
# 에피소드 스토리
에피소드: {번호}  제목: {제목}

## 요약 (3줄)

## 장면 목록
### 장면 N — {장소} / {분위기}
- 등장 캐릭터:
- 핵심 사건:
- 감정 흐름:
- 패널 힌트:
- 대사 분량: (많음/보통/없음)

## 감정 곡선
[시작] → [전환점] → [클라이맥스] → [엔딩]

## 다음 에피소드 떡밥
```
